# KernelSU Modules Repository

- Modules Website: https://modules.kernelsu.org
- Submit Module: https://modules.kernelsu.org/submission

## API Reference
- All Modules: `GET https://modules.kernelsu.org/modules.json`
- Module: `GET https://modules.kernelsu.org/module/[moduleId].json`
- Search Index: `GET https://modules.kernelsu.org/search-index.json`
